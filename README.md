# JavaScript Roadmap Practice

A collection of small JavaScript exercises solved as part of the
[roadmap.sh](https://roadmap.sh) JavaScript track. Each folder is a
self-contained project with its own README, including how to run it
and a link to the original challenge.

## How to Run Any Project

\`\`\`bash
git clone https://github.com/Danbaba1/javascript-roadmap-practice.git
cd javascript-roadmap-practice/<project-folder>
node index.js
\`\`\`

Replace `<project-folder>` with the folder name of the project you
want to run (e.g. `cart-total`, `password-rule-checker`).
